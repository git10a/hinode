# HINODE SEOプレイブック

作成日: 2026-06-12

サイト側の実装（コースガイド・FAQ・llms.txt）とセットで進める、サイト外の施策の実行手順をまとめたドキュメント。上から順に費用対効果が高い。

---

## 1. Googleビジネスプロフィール登録（所要: 30分 + 確認待ち）

「ランニングクラブ 渋谷」「ランニングサークル 東京」などのローカル検索で地図枠（ローカルパック）に出るための施策。被リンクとしても機能する。

1. https://business.google.com/ で「HINODE」を作成
2. カテゴリ: 「スポーツクラブ」または「アスレチッククラブ」
3. ビジネス所在地: 非公開（非店舗型）にし、サービス提供地域を「渋谷区・千代田区・目黒区」に設定
4. 営業時間: 水 6:00–7:00 / 木 6:00–7:00 / 日 7:15–8:30 として登録
5. ウェブサイト: https://hinode-run.com/ 、予約リンク欄に https://hinode-run.com/first-run
6. 説明文はトップページのdescriptionをベースに。「予約不要・参加無料・1人参加歓迎」を必ず含める
7. 開催ごとの写真を月数枚アップ（更新シグナルになる）
8. 参加者に口コミを依頼（5件あると表示が安定する）

## 3. デジタルPR（所要: 半日でメール送付まで）

プレスキット（/press）があるので、あとは送るだけ。掲載1本の被リンクは記事数十本分の価値がある。

### 売り込むストーリー案
- 「日の出とともに走る」朝5時台の東京の写真（ビジュアルが強い）
- 「競わないランニングコミュニティ」2025年開始で毎週連続開催継続中という数字
- 「朝活の最前線」若手社会人が飲み会ではなく朝ランで集まる、という時流ネタ

### 候補メディア
| メディア | 切り口 |
|---|---|
| シブヤ経済新聞 | 代々木公園×地域ネタ。個人コミュニティの掲載実績が多い |
| Time Out Tokyo | 「東京の朝の過ごし方」系特集 |
| RUNNET / ランナーズ | ランニングコミュニティ紹介 |
| Runtrip Magazine | コース×コミュニティ |
| 各種朝活・ライフスタイル系Webメディア | 「朝活」文脈 |

### ピッチメールの型
件名: 【取材のご案内】日の出とともに走る東京の朝ランコミュニティ「HINODE」
- 1段落目: 何者か（毎週、日の出前に集まって走る無料コミュニティ。2025年11月から雨天を除き毎週開催）
- 2段落目: なぜ今か（朝活トレンド・競わない緩いコミュニティの広がり）
- 3段落目: 提供できるもの（早朝の写真素材・主催者インタビュー・参加者の声）+ プレスキットURL

## 4. ブログ運用（AI併用・月4本ペース）

### 制作フロー
1. 下のクラスタ表からキーワードを1つ選ぶ
2. Claudeに「このキーワードで、HINODEのブログ記事の下書きを。トーンはhinode-run.com/manifestoに合わせる」と依頼（このリポジトリで作業すれば文体・事実関係を参照できる）
3. **自分にしか書けない一次情報を必ず足す**: 実際の開催の写真、参加者の発言、自分の体験。ここがAI記事との差別化でありSEOの生命線
4. microCMSに入稿。記事末尾は必ず /first-run への誘導で締める

### キーワードクラスタ表（優先度順）

**A. 不安解消系（参加意図に最も近い）**
| キーワード | 記事案 |
|---|---|
| ランニングサークル 一人 参加 | 1人で参加した人がその日どう過ごしたか、実例で答える |
| ランニングサークル 初心者 怖い | 「初参加の不安全部答えます」体験談形式 |
| 走るの遅い サークル | 「会話ペース」とは実際何分/kmなのか正直に書く |
| ランニング 仲間 作り方 社会人 | 社会人がラン仲間を作る選択肢を比較して紹介 |

**B. 朝ラン習慣系（潜在層が大きい）**
| キーワード | 記事案 |
|---|---|
| 朝ラン 続かない | 「意志ではなく約束で続ける」HINODE思想そのもの |
| 朝ラン 効果 | 体験ベース+一般的な知見。誇張しない |
| 朝ラン 何時から | 日の出時刻ページ（/sunrise）に内部リンク |
| 朝ラン 冬 暗い 安全 | 冬の開催で実際にやっている工夫 |
| 朝ラン 食前 食後 | 参加者の実例を集める |

**C. 場所系（コースガイドの補完）**
| キーワード | 記事案 |
|---|---|
| 皇居ラン 初めて | コースガイド(/courses/kokyo)へリンクするレポート記事 |
| 代々木公園 朝 雰囲気 | 写真多めの朝レポート |
| 中目黒 朝活 | ラン+カフェの朝活コース紹介 |

### 月次ルーチン
- 月1回、Search Consoleの「表示回数が多いのに順位が低い」クエリを確認し、既存記事に追記するか新記事のネタにする

## 5. 計測（KPI設計)

- **最重要プロキシKPI**: /first-run への自然検索流入数（GA4で landing page = /first-run × organic）
- Search Consoleで監視するクエリ群:
  - 場所系: 「皇居 ラン」「代々木公園 ランニング」「目黒川 ランニング」
  - 仲間系: 「ランニングサークル 東京」「朝ラン 東京」「朝活 ランニング」
  - ブランド: 「HINODE ラン」「ヒノデ ランニング」
- コースガイド公開後2〜4週でSearch Consoleの「ページ」レポートに /courses/* が載り始めるのが正常。載らなければサイトマップ送信を確認
